# OpenClaw WSL 安装指南

本仓库提供在 Windows 11 WSL 环境下安装和配置 OpenClaw 的完整教程，整合了 WSL 环境配置、OpenClaw 安装、国内 API 配置及局域网访问设置。

## 内容

- [WSL 安装与 OpenClaw 部署指南](wsl-installation.md)

## 涵盖内容

1. **WSL 一键安装** - Windows 11 环境下快速部署 WSL2
2. **OpenClaw 安装** - 官方一键脚本安装与初始化配置
3. **国内 API 配置** - MiniMax、智谱 GLM 等国内模型配置
4. **局域网访问** - 配置 Gateway 支持局域网访问
5. **网络优化** - 镜像模式配置与防火墙设置
